Animal Revolt!

Game Development

Animal Revolt!

3D FPS game created using Unity


Awarded Third in the 6th NUS School of Computing Term Project Showcase.

Animal Revolt! is a mixture of first-person and third-person shooter game created using Unity featuring a variety of ammunitions and enemy AI. The game features three playable levels of differing objectives and difficulty.

Players are challenged by tough enemy AI and the need for resource management due to limited ammunition. With 10 different types of ammunition with different effects, players can choose to complete each level in many creative ways.

View website at jiajieyong.github.io/Animal-Revolt

Gently

Real-time game streaming service with server load balancing


Gently is a streaming service similar to Twitch and Youtube Live Streaming. It uses a Content Delivery Network architecture for end-to-end real time streaming.

Streamers can choose to stream with either popular video broadcasting tools like Open Broadcaster Software or with our own lightweight screen recorder that streams directly to our service.


View source at github.com/oswellchan/Gently

Cognizant Tower Challenge

Web Development

Cognizant Tower Challenge

Cognizant Tower Challenge (CTC) is a proof of concept project for Congnizant's HR department.

CTC is a web application that gamifies the initial technical interview. Applicants are to sign up the website and take part in the different challenges.

The HR department will then be able to collect the neccessary data and pick out the top perfomring applicants. This helps to speed up the choosing process and allow HR to analysis the applicants more easily
